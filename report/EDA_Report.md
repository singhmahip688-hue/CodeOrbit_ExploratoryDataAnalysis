# Exploratory Data Analysis Report

## Dataset

Employee Dataset

## Data Cleaning

- Removed duplicate records.
- Converted Age, Salary, and Experience columns to numeric format.
- Filled missing Age values using the median.
- Filled missing Salary values using the mean.
- Filled missing Experience values using the median.
- Filled missing Department values with "Unknown".

## Visualizations

### Histogram

The salary distribution shows that most employees earn between ₹45,000 and ₹65,000.

### Bar Chart

The IT department has the highest number of employees, followed by HR, Finance, and Marketing.

### Scatter Plot

A positive relationship exists between age and salary. Employees with greater age generally have higher salaries.

## Outliers

No major salary outliers were detected using the IQR method.

## Conclusion

The dataset was successfully cleaned and analyzed. The visualizations provide useful insights into employee demographics, salary distribution, and departmental composition.
