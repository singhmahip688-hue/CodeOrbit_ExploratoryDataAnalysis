# Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on an employee dataset using Python.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn

## Project Tasks

- Load dataset
- Clean missing values
- Remove duplicates
- Convert incorrect data types
- Perform statistical analysis
- Create visualizations
- Identify trends and outliers

## Visualizations

- Salary Histogram
- Department Bar Chart
- Age vs Salary Scatter Plot
